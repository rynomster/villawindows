# Villa Windows Design System

This design system defines the visual language for the Villa Windows web presence, focusing on heritage character preservation and modern reliability.

## Colors

- **Primary Blue** (#0056b3): Used for headers, buttons, and call-to-actions. Represents trust and professionalism.
- **Deep Navy** (#333333): Primary text color and footer background.
- **Light Gray** (#f4f4f4): Background for hero sections and subtle dividers.
- **Border Gray** (#dddddd): Used for cards and input borders.
- **White** (#ffffff): Main page background and text on dark backgrounds.

## Typography

- **Primary Font**: Arial, sans-serif (Universal, clean, and reliable).
- **Headings**:
  - H1: Hero sections, 2.5rem, bold.
  - H2: Section headers, 2rem, bold.
  - H3: Component headers, 1.5rem, bold.
- **Body**: 1rem (16px), 1.6 line-height for optimal readability.

## Spacing

- **Base Unit**: 8px.
- **Section Padding**: 40px - 60px.
- **Grid Gap**: 20px.

## Components

### Buttons
- **Style**: Filled Primary Blue, white text.
- **Padding**: 10px 20px.
- **Border Radius**: 5px.
- **Usage**: "Free Estimate", "Learn More", form submission.

### Cards
- **Style**: White background, 1px border (#ddd), 8px radius.
- **Shadow**: None (Keeping it clean and flat).
- **Layout**: Image at top, followed by 15px margin, then text.

### Hero Section
- **Background**: High-quality imagery (timber windows/craftsmanship) with a dark overlay (rgba(0,0,0,0.5)) if text is light.
- **Alignment**: Centered text and CTA.

## Guidelines

- **Character**: Always emphasize the "original look" and "heritage" of the timber joinery.
- **Imagery**: Use real-world photos of finished projects and craftsmanship. Avoid generic stock photos where possible.
- **Tone**: Professional, knowledgeable, and respectful of traditional architecture.
